## miniproject 소개
- 기업이 휴가를 효과적으로 관리할 수 있게 도와줍니다.
- 직원 버전과 관리자 버전을 둘 다 제공하여 사용자가 선택한 뒤 선택한 서비스를 제공합니다.
- 직원에게 신청, 삭제, 조회 등의 기능을 제공하며 관리자에게는 처리, 조회 등의 기능을 제공합니다.
